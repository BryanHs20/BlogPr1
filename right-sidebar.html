¿Qué es la conexión libre?

Internet Libre

Este se puede catalogar como libre, cuando estamos frente a un sistema descentralizado y autosustentable. La descentralización puede tomarse como ejemplo lo que ocurre en las trasmisiones streaming donde es una emisión directa sin intermediarios.

Internet Abierto

Es un sistema abierto cuando existe transparencia en cada una de las comunicaciones, permitiendo una colaboración grupal de diferentes puntos, compartiendo cualidades e ideas.

¿Cuál es el objetivo de tener una Internet libre y abierta y qué ventajas tiene?

Dejando a un lado las desventajas, podremos decir que un Internet libre y abierto nos permite conectarnos y apoyar todos los sectores, incluyendo entre ellos el área de salud humana.

Ventajas

  1. Sus ventajas se basan en la comunicación, donde no existen terceros ni organismos que interfieran en ella, situación que conecta al mundo sin necesidad de fronteras o algún tipo de discriminación.
  2. Un sistema abierto proporciona al usuario final/integrador la opción de crear un sistema óptimo sin las restricciones de un único fabricante. No es así un sistema cerrado donde, además, los usuarios finales se ven atados a productos patentados de un fabricante sin estándares abiertos.

¿Qué es la conexión a traves de redes?

En las redes informáticas, los dispositivos intercambian datos entre sí mediante conexiones (enlaces de datos) a través de nodos. Estos enlaces de datos se establecen a través de medios como cables, cables ópticos o medios inalámbricos como Wi-Fi.

¿Qué es el modelo OSI?

Una de las principales referencias es el modelo OSI (Open System Interconnection), que no es una arquitectura de red, sino una normativa formada por siete capas que definen las diferentes fases por las que deben pasar los datos para viajar de un dispositivo a otro sobre una red de comunicaciones.

Partes del modelo OSI

El modelo OSI usa capas para dar una descripción visual de lo que está sucediendo con un sistema de red en particular. Las 7 capas son:

  1. Aplicación
  2. Presentación
  3. Sesión
  4. Transporte
  5. Red
  6. Enlace de Datos

Clasificación de los tipos de diseño de redes

Por razones históricas, la industria se refiere a casi todo tipo de diseño como algún tipo de red de área. Los tipos más comunes son:

  1. LAN: Red de área local
  2. WAN: Red de área amplia
  3. WLAN: Red de área local inalámbrica
  4. MAN: Red de área Metropolitana
  5. SAN: Red de área de almacenamiento
