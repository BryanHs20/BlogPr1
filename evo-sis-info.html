1950 - 1960: procesamiento de datos electrónicos, sistema de procesamiento de transacciones

A lo largo de este lapso, el papel del sistema de información era primordialmente hacer ocupaciones como el procesamiento de transacciones, el mantenimiento de registros y la contabilidad. SI se usa primordialmente para el procesamiento electrónico de datos. Además conocido como procesamiento de información o procesamiento automático de datos. Un Sistema de procesamiento de transacciones (TPS) ha sido el primer sistema computarizado desarrollado para procesar datos comerciales. TPS principalmente informatiza o automatiza los procesos manuales existentes para un procesamiento más veloz, un mejor servicio al comprador y precios administrativos más bajos. TPS además se sabe como procesamiento de transacciones o procesamiento en tiempo real.

1960 a 1970: Sistemas de información gerencial

Durante esta era, el rol de SI evolucionó desde TPS hasta Management Information Systems (MSI) o Sistemas de información gerenciales. MSI procesa los datos en informes informativos útiles y proporciona a los gerentes las herramientas para organizar evaluar y administrar de manera eficiente los departamentos dentro de una organización.

Por lo general, MSI crea 3 tipos básicos de información que son:

  • Los informes detallados de información típicamente confirman las actividades de procesamiento de transacciones. Un informe de pedido detallado es un ejemplo de informe detallado.
  • La información resumida instituye datos en un formato que un sujeto puede comprobar veloz y de forma sencilla.
  • Los informes de excepciones ayudan a los gerentes a ahorrar tiempo pues no deben buscar en un informe descriptivo las excepciones.

    • 1970 a 1980: Sistemas de soporte de decisiones

    El papel se centró en el "sistema interactivo con base en pcs" para contribuir a los causantes de la toma de elecciones a solucionar inconvenientes. Este nuevo papel de los sistemas de información para dar soporte ad-hoc interactivo para el proceso de toma de elecciones a los gerentes y otros expertos de negocios se llama Sistemas de Soporte de Decisiones (DSS). DSS sirve el nivel de planificación, gestión y operaciones de una organización, por lo general, la alta dirección.

    DSS usa datos de fuentes internas y/o externas. Las fuentes internas de datos tienen la posibilidad de integrar datos de inventario, ventas, construcción o financieros de la base de datos de una organización. Las fuentes externas podrían integrar costos, tasas de interés, población o tendencias. Los gerentes utilizan DSS para manipular los datos y favorecer con las decisiones.

    1980 a 1990: Sistemas de información ejecutivo

    Este período dio lugar a la informática departamental debido a que muchas organizaciones compran su propio hardware y software para satisfacer sus necesidades departamentales. En vez de aguardar el apoyo indirecto del departamento de servicio corporativo centralizado, los empleados podrían utilizar sus propios recursos para respaldar sus requisitos laborales. Además, los elevados ejecutivos no utilizaban DSS ni MSI, por lo cual se desarrollaron sistemas de información ejecutivo (ESI) o sistemas de soporte ejecutivo (ESS).

    ESI da servicios de toma de elecciones a los ejecutivos por medio del abastecimiento de información interna y externa importante para consumar las metas estratégicos de la organización. Los ejemplos del ESI son sistemas para facilitar la entrada a las ocupaciones de todos los participantes, desarrollos económicos para respaldar la organización estratégica y la exploración del funcionamiento del comercio.

    1990 a 2000: Sistemas de gestión del conocimiento

    Durante esta era, el rápido crecimiento de las intranets, extranets, internet y otras redes globales interconectadas cambiaron drásticamente las capacidades de SI en los negocios. Se hizo posible circular el conocimiento a diferentes partes del mundo independientemente del tiempo y el espacio. Este lapso además vio una aparición de sistemas de planeación de recursos empresariales (ERP). ERP es una manera específica de organización de un sistema de información estratégica que añade todos los elementos de una organización, incluidas la construcción, las ventas, la administración de recursos, la planeación de recursos humanos y el marketing.

    Sistemas expertos (ES) y sistemas de gestion de conocimiento (KMS) interconectados entre sí. Los sistemas expertos (ES) son un sistema informático que copia la función de toma de decisiones de los expertos humanos. El sistema de gestion de conocimiento (KMS) es un sistema de TI que almacena y recupera entendimiento para auxiliar la construcción, organización y diseminación del entendimiento empresarial en la organización. ES usa datos de los Sistemas de Gestion de Conocimientos para crear un resultado deseable del sistema de información, ejemplificando, un sistema de aprobación de solicitudes de préstamo.

    2000 - Presente: E-Business

    Internet y las tecnologías y aplicaciones en relación cambiaron la manera en que operan las organizaciones y los individuos trabajan. Las funcionalidades de los sistemas de información en este lapso siguen siendo las mismas que hace 50 años: mantenimiento de registros, administración de informes, procesamiento de transacciones, administración de soporte y procesos de administración de la organización. Muchas organizaciones utilizan tecnologías de Internet y procesos de comercio habilitados para la web para producir aplicaciones innovadoras de negocio electrónico.
